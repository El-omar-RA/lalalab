# LalaLab — Mushroom Growing Supplies & Courses

E-commerce + education website for mushroom cultivation:
- Shop: supplies (spawn, kits, substrates, tools)
- Courses: structured learning (beginner → advanced)
- Learn: articles/guides and troubleshooting resources

## Tech Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Stripe Checkout (products + courses)
- Headless CMS for content (Sanity recommended; can swap)
